# Murls Firebase
Murls Firebase is a URL Shortener that runs on Firebase for free.

## Features:
* Runs for free**
* Easy to install
* Custom Domain
* Free SSL
* Open source
* Analytics Included
  * Clicks (Mobile & Desktop)
  * Referrers
  * Country
  * Platform (Ex: Windows, Mac)
* Adding URLs is protected by Authentication
